\textbackslash{}rst A dynamic version of {\ttfamily fmt\+::format\+\_\+arg\+\_\+store}. 

It\textquotesingle{}s equipped with a storage to potentially temporary objects which lifetimes could be shorter than the format arguments object.

It can be implicitly converted into {\ttfamily $\sim$fmt\mbox{\hyperlink{classbasic__format__args}{basic\+\_\+format\+\_\+args}}} for passing into type-\/erased formatting functions such as {\ttfamily $\sim$fmt\+::vformat}. \textbackslash{}endrst
