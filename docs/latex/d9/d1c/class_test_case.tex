\doxysubsection{Detailed Description}
A single test object. 

This class is used to implement a simple test case\+: define a subclass that overrides the run\+Test method.

You don\textquotesingle{}t usually need to use that class, but \mbox{\hyperlink{class_test_fixture}{Test\+Fixture}} and \mbox{\hyperlink{class_test_caller}{Test\+Caller}} instead.

You are expected to subclass \mbox{\hyperlink{class_test_case}{Test\+Case}} is you need to write a class similiar to \mbox{\hyperlink{class_test_caller}{Test\+Caller}}.
