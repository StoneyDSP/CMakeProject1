\doxysubsection{Detailed Description}
Record of a failed \mbox{\hyperlink{class_test}{Test}} execution. 

A \mbox{\hyperlink{class_test_failure}{Test\+Failure}} collects a failed test together with the caught exception.

\mbox{\hyperlink{class_test_failure}{Test\+Failure}} assumes lifetime control for any exception passed to it.
