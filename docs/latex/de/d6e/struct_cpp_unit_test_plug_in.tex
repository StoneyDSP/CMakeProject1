\doxysubsection{Detailed Description}
\mbox{\hyperlink{class_test}{Test}} plug-\/in interface. 

This class define the interface implemented by test plug-\/in. A pointer to that interface is returned by the function exported by the test plug-\/in.

Plug-\/in are loaded/unloaded by \mbox{\hyperlink{class_plug_in_manager}{Plug\+In\+Manager}}. When a plug-\/in is loaded, \mbox{\hyperlink{struct_cpp_unit_test_plug_in_aec670330e7fced26c2a66b1dcd56edc0}{initialize()}} is called. Before unloading the plug-\/in, the \mbox{\hyperlink{class_plug_in_manager}{Plug\+In\+Manager}} call \mbox{\hyperlink{struct_cpp_unit_test_plug_in_a8628d2026e76c58f715e17af88f77458}{uninitialize()}}.

\mbox{\hyperlink{struct_cpp_unit_test_plug_in_aad8038dc72d0f9798379937fe5692c97}{add\+Listener()}} and \mbox{\hyperlink{struct_cpp_unit_test_plug_in_a8f36157014b515d38efbc8ab67923d85}{remove\+Listener()}} are called respectively before and after the test run.

\mbox{\hyperlink{struct_cpp_unit_test_plug_in_a547cfddd0513dc9182721f723e27d9e3}{add\+Xml\+Outputter\+Hooks()}} and \mbox{\hyperlink{struct_cpp_unit_test_plug_in_a045727ad9658525838b0b9157065fbcd}{remove\+Xml\+Outputter\+Hooks()}} are called respectively before and after writing the XML output using a \mbox{\hyperlink{class_xml_outputter}{Xml\+Outputter}}.
