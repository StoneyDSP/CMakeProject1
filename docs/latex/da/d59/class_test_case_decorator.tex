\doxysubsection{Detailed Description}
Decorator for \mbox{\hyperlink{class_test}{Test}} cases. 

\mbox{\hyperlink{class_test_case_decorator}{Test\+Case\+Decorator}} provides an alternate means to extend functionality of a test class without subclassing the test. Instead, one can subclass the decorater and use it to wrap the test class.
