\doxysubsection{Detailed Description}
A XML Document. 

A \mbox{\hyperlink{class_xml_document}{Xml\+Document}} represents a XML file. It holds a pointer on the root \mbox{\hyperlink{class_xml_element}{Xml\+Element}} of the document. It also holds the encoding and style sheet used.

By default, the XML document is stand-\/alone and tagged with enconding \char`\"{}\+ISO-\/8859-\/1\char`\"{}.
