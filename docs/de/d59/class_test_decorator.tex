\doxysubsection{Detailed Description}
Decorator for Tests. 

\mbox{\hyperlink{class_test_decorator}{Test\+Decorator}} provides an alternate means to extend functionality of a test class without subclassing the test. Instead, one can subclass the decorater and use it to wrap the test class.
