\doxysubsection{Detailed Description}
Collects test result. 

A \mbox{\hyperlink{class_test_result_collector}{Test\+Result\+Collector}} is a \mbox{\hyperlink{class_test_listener}{Test\+Listener}} which collects the results of executing a test case. It is an instance of the Collecting Parameter pattern.

The test framework distinguishes between failures and errors. A failure is anticipated and checked for with assertions. Errors are unanticipated problems signified by exceptions that are not generated by the framework. \begin{DoxySeeAlso}{See also}
\mbox{\hyperlink{class_test_listener}{Test\+Listener}}, \mbox{\hyperlink{class_test_failure}{Test\+Failure}}. 
\end{DoxySeeAlso}
