\doxysubsection{Detailed Description}
Represents a source line location. 

Used to capture the failure location in assertion.

Use the CPPUNIT\+\_\+\+SOURCELINE() macro to construct that object. Typically used when writing an assertion macro in association with \mbox{\hyperlink{struct_asserter}{Asserter}}.
