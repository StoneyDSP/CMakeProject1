\doxysubsection{Detailed Description}
Generic test runner. 

The \mbox{\hyperlink{class_test_runner}{Test\+Runner}} assumes ownership of all added tests\+: you can not add test or suite that are local variable since they can\textquotesingle{}t be deleted.
