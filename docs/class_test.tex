\doxysubsection{Detailed Description}
Base class for all test objects. 

All test objects should be a subclass of \mbox{\hyperlink{class_test}{Test}}. Some test objects, \mbox{\hyperlink{class_test_case}{Test\+Case}} for example, represent one individual test. Other test objects, such as \mbox{\hyperlink{class_test_suite}{Test\+Suite}}, are comprised of several tests. ~\newline


When a \mbox{\hyperlink{class_test}{Test}} is run, the result is collected by a \mbox{\hyperlink{class_test_result}{Test\+Result}} object.
