\doxysubsection{Detailed Description}
Outputs a \mbox{\hyperlink{class_test_result_collector}{Test\+Result\+Collector}} in XML format. 

Save the test result as a XML stream.

Additional datas can be added to the XML document using \mbox{\hyperlink{class_xml_outputter_hook}{Xml\+Outputter\+Hook}}. Hook are not owned by the \mbox{\hyperlink{class_xml_outputter}{Xml\+Outputter}}. They should be valid until destruction of the \mbox{\hyperlink{class_xml_outputter}{Xml\+Outputter}}. They can be removed with \mbox{\hyperlink{class_xml_outputter_ac4659712ab6b0f168119a22fbdb70255}{remove\+Hook()}}.
