class Exception\+Test\+Case\+Decorator$<$ Expected\+Exception $>$}
Expected exception test case decorator. 

A decorator used to assert that a specific test case should throw an exception of a given type.

You should use this class only if you need to check the exception object state (that a specific cause is set for example). If you don\textquotesingle{}t need to do that, you might consider using \mbox{\hyperlink{group___writing_test_fixture_gaca8eeb6f60714baade6cbfd185868c40}{CPPUNIT\+\_\+\+TEST\+\_\+\+EXCEPTION()}} instead.

Intended use is to subclass and override \mbox{\hyperlink{class_exception_test_case_decorator_a0a45d6182ab00e9013593311845e50bb}{check\+Exception()}}. Example\+:
